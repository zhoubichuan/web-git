# 四.高级命令

## 1.变基操作

先拉取远程代码合并到本地代码中，然后将开发的代码合并到dev分支中

```sh
git checkout dev # 切换到dev
```

```sh
git fetch origin dev # 拉取远程dev
```

```sh
git rebase origin/dev # 变基
```

## 2.挑拣提交

将`feature`分支的某些提交 cherry-pick 到`dev`

```sh
git checkout dev # 切换到dev
```

```sh
git cherry-pick a1b2c3d # 挑拣提交
```
