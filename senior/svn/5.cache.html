# 四.常用的cmd命令

# 删除文件夹操作

文件目录如下

  • c
    • admin
      • my

# 1.删除 admin 下面的所有文件和文件夹

在 c 目录下面打开 cmd 命令窗口

rmdir /s/q admin
1
  • /s 是代表删除所有子目录跟其中的档案。
  • /q 是不要它在删除档案或目录时，不再问我 Yes or No 的动作。

# 2.删除指定目录下的文件

rmdir /s/q c:\admin\my
1

删除 c 盘下面的 admin 目录下面的 my
